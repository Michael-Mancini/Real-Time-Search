Real Time Search to the cdnjs API with Observables and RxJS


Search Service

Import HttpClient, Observable, and rxjs debounceTime, distinctUntilChanged, and switchMap operators.

The search method takes an observable of strings and uses the rxjs operators to limit the amount of requests.

'debounceTime' takes a milliseconds parameter and waits until that time passes with no changes before letting the data pass through. I used 250ms.

'distinctUntilChanged' ensures that only distinct or different data passes through.

'switchMap' combines observable strings recieved from the searchEntries method into one variable.

The searchEntries method makes the API get request with the search term after passing through the rxjs operators. This returns an observable and is automatically mapped to json by Angular HttpClient.


Component

Initialize searchTerm variable as a new rxjs Subject and a results variable.

In the constructor, call the searchService 'search' method passing in the subject and subscribing to it. In the subscription assign the local results variable to the search results.


Template

Use the keyup event handler on the input tag to call the .next() method on the searchTerm subject, passing in the value.

Iterate over the first 10 results using ngFor and the slice pipe.

Lastly, use the Roboto font and apply some basic styles.
